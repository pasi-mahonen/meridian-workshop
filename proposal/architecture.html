Data flow

User interactions update Vue's reactive filter state via useFilters. Components watch those refs and call api.js, which passes warehouse/category/budget as query params to the backend. FastAPI applies apply_filters() and returns filtered JSON. No server-side state is mutated.

Known constraints

No persistence layer. All data lives in mock_data.py and resets on server restart. Task CRUD endpoints were wired in the previous vendor's iteration but write operations are not yet exposed. Scaling to real data will require a database migration (e.g. PostgreSQL + SQLAlchemy).
